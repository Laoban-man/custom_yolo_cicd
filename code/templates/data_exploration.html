Handling the data

Downsampling

In order to work with limited resources (NVIDIA GTX 1660 Ti 6GB), the images can be downsampled to reduce the number of anchors to inspect per image.

Standardized annotations

Many deep learning object detection frameworks function with specific annotations formats so the dots identifying individual instances must be detected and converted automatically to bounding box formats (COCO or YOLO).

Augmentation

Images can be augmented with the CLODA/Albumentations/Detectron2/Yolov5 packageS to increase the number of training images (flipping, cropping, add blur or sharpen) while maintaining the annotations
